Recover - C

This was a project I started back in March 2020 when I learned Pointers in C (Programming Language).

When we'll run this program, it will open a memory card given from user, and read all of the data from the memory card.


After reading the memory card into the memory, find all of the images that has these specific first three bytes (0xff 0xd8 0xff). When find these specfic bytes, then we'll know this is a starting of the image file that we have to recover.

When we'll find these specific bytes program will create a new file and copy all of the bytes to the new file.

I worked on this project for a week.
